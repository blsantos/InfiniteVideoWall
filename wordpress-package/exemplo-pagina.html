Testemunhos Contra o Racismo

Como Contribuir

Você também pode compartilhar sua experiência:

  1. Use o botão "Upload" na interface acima
  2. Grave seu testemunho (máximo 60 segundos)
  3. Preencha os dados demográficos (opcional)
  4. Envie para moderação

Sobre o Projeto

O projeto Reparações Históricas tem como objetivo amplificar vozes que sofreram racismo, criando uma plataforma segura para compartilhamento de experiências e conscientização social.
